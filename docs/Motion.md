# 运动积木<a id="运动积木"></a>
运动是什么？<br>

<b>当角色在一段时间内位置或者方向发生了改变，这个角色就在运动。</b><br>

动作指令用于调整角色在舞台中的位置和方向，获取角色当前的坐标值和方向值。<br>

## 移动（10）步
<img src=images\motion\1.png height = 100px><br>
<b>功能：</b>使角色移动设置步数，没有移动过程。<br>
<b>参数：</b>[椭圆形参数](Code.md#section2)，1步代表1个像素。<br>
<b>效果：</b>
* 参数大于0，向角色向面向的方向移动。<br>
* 参数小于0，向角色向相反方向移动。<br>

## 右转（15）度
<img src=images\motion\2.png height = 100px><br>
<b>功能：</b>使角色顺时针旋转参数设置的度数，没有旋转过程。<br>
<b>参数：</b>[椭圆形参数](Code.md#section2)，旋转的度数。<br>

## 左转（15）度
<img src=images\motion\3.png height = 100px><br>
<b>功能：</b>使角色逆时针旋转参数设置的度数，没有旋转过程。<br>
<b>参数：</b>[椭圆形参数](Code.md#section2)，旋转的度数。<br>

## 移到（随机位置）
<img src=images\motion\4.png height = 100px><br>
<b>功能：</b>使角色移动到参数设置的位置，没有移动过程。<br>

<b>参数：</b>[椭圆形下拉参数](Code.md#section1)，默认选项有随机位置和鼠标指针，添加角色后，选项增加角色名称。<br>

随机位置，移动到舞台中的任意位置，每次移动的位置不相同。<br>

鼠标指针，移动到鼠标在舞台中的位置。<br>

角色名称，移动到所选角色的坐标位置。可以使用变量作为参数，变量值设置为角色名称。<br>

返回值：无


## 移到x：(0) y：(0)
<img src=images\motion\5.png height = 100px><br>
<b>功能：</b>使角色移到设置的坐标位置，没有移动过程。<br>

<b>参数：</b>[椭圆形参数](Code.md#section2)，代表角色移动的目标坐标。<br>

积木默认的坐标值会随着角色在舞台中的位置发生改变。<br>

返回值：无

## 在（1）秒内滑行到(随机位置)
<img src=images\motion\6.png height = 100px><br>
<b>功能：</b>使角色移动到参数位置，展示移动过程，移动花费的时间为设置的时间参数。<br>

时间<b>参数：</b>椭圆形参数。代表角色从当前位置移动到目标位置花费的时间。<br>

目标位置<b>参数：</b>椭圆形下拉参数，同移到([随机位置])积木。<br>

返回值：无


## 在（1）秒内滑行到x：(0) y：(0)
<img src=images\motion\7.png height = 100px><br>
<b>功能：</b>使角色移动到设置的坐标位置，展示移动过程，移动花费的时间为设置的时间参数。<br>

时间<b>参数：</b>椭圆形参数。代表角色从当前位置移动到目标位置花费的时间。<br>

目标坐标<b>参数：</b>椭圆形参数，同移到x：(0) y：(0)积木。<br>

## 面向（鼠标指针）
<img src=images\motion\16.png height = 100px><br>
<b>功能：</b>使角色方向变为指向参数选项。<br>

<b>参数：</b>椭圆形下拉参数，默认选项有鼠标指针，添加角色后，选项增加角色名称。<br>

鼠标指针为角色方向指向鼠标指针停留位置。<br>

角色名称为角色方向指向被选中角色的坐标点。可以使用变量作为参数，变量值设置为角色名称。<br>

返回值：无


## 将x坐标增加（10）
<img src=images\motion\17.png height = 100px><br>
<b>功能：</b>将角色移动到x坐标增加后的新坐标位置，y坐标不变，没有移动过程。控制角色左右移动。<br>

<b>参数：</b>椭圆形参数。<br>

数字大于0，角色水平向右移动。数字小于0，角色水平向左移动。<br>

返回值：无

## 碰到边缘就反弹
<img src=images\motion\10.png height = 100px><br>
<b>功能：</b>角色碰到舞台边缘时，发生反弹，类似于光在镜面的反射。<br>

当碰撞到上下边缘，0 ≤ 当前方向 ≤ 180时，反弹后方向 = 180 - 当前方向。<br>

当碰撞到上下边缘，-180 ＜ 当前方向 ＜ 0时，反弹后方向 = -(180 + 当前方向)。<br>

当碰撞到左右边缘时，反弹后方向 = -当前方向

<b>参数：</b>无

返回值：无
